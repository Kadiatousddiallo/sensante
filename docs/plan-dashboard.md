# Plan Dashboard – SénSanté

## 🎯 Objectif
Le dashboard permet de visualiser rapidement les informations importantes sur les patients afin d’aider à la prise de décision.

---

## 📊 Métriques proposées
- Nombre total de patients
- Nombre de patients par région
- Répartition des patients par sexe (H/F)
- Âge moyen des patients
- Nombre de nouveaux patients (récents)

---

## 📈 Graphiques
- Diagramme en barres : patients par région
- Diagramme circulaire (camembert) : répartition homme / femme
- Histogramme : répartition par âge
- Courbe : évolution du nombre de patients

---

## 🧩 Structure du dashboard
- En haut : cartes avec statistiques (total patients, âge moyen…)
- Au centre : graphiques principaux
- En bas : liste récente des patients

---

## 🛠️ Technologies envisagées
- React (frontend)
- Recharts ou Chart.js pour les graphiques
